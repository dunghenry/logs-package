### Build a NPM package using Typescript 
## Usage
### Script

```js
import {Log} from 'logs-package';
```
###
### Log type error

```js
 Log.error("Helllo")
```
###

### Log type success

```js
 Log.success("Helllo")
```
###

### Log type info
```js
 Log.info("Helllo")
```
###
### Log type warn

```js
 Log.warn("Helllo")
```
